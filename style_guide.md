# GoRan AI Brand Visual Style & Color Palette Guide

Use this specification for all image generations, layouts, and styles.

## Color Palette

| Usage | CSS Variable / RGB | HEX | Visual Preview |
|---|---|---|---|
| **Primary Background** | `rgb(255, 255, 255)` / `rgb(250, 246, 240)` | `#FFFFFF` / `#FAF6F0` | Pure white or warm off-white |
| **Grid Lines** | `rgb(240, 238, 234)` | `#F0EEEA` / `#EBE6DC` | Ultra light grey/cream thin lines |
| **Primary Typography** | `rgb(17, 17, 17)` | `#111111` | Bold carbon black |
| **Secondary Typography** | `rgb(102, 102, 102)` | `#666666` | Clean slate grey |
| **Brand Accent (Highlight)**| `rgb(245, 166, 35)` | `#F5A623` | Warm golden amber/yellow |

## Typography & Design Elements
- **Layout Grid**: Thin vertical grid lines spaced evenly to guide structural alignment.
- **Minimalist Aesthetics**: Use simple black vector strokes, solid colors, and plenty of negative space.
- **Accents**: Highlight key words, steps, or metrics with the brand **amber/yellow** color.
- **Logos**: Use the GoRan AI logo (`https://res.cloudinary.com/dvwpxb2oa/image/upload/v1781607296/Logo_sw7cqh.png`) in headers or covers.
